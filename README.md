## Cours Développement d’interfaces 4 — CMS

Dans ce cours, vous apprendrez à construire des sites avec des outils avancés : des CMS (Content Management System). 

Ces outils permettent de:

- Construire un site en utilisant une interface visuelle.
- Gérer des contenus et les organiser (par exemple créer des catégories, des groupes, des hiérarchies de contenus)
- Garder une séparation entre le code et la gestion des contenus.

Le CMS que nous utiliserons principalement est WordPress (augmenté de l'interface Bricks). Les raisons de ce choix:

- WordPress est le CMS le plus largement utilisé dans le monde. Il y a de grandes chances que vous le rencontrerez dans votre vie professionnelle d'IMD.
- WordPress est open-source et gratuit. Vous pouvez donc l'utiliser sans être forcé de payer une licence ou un abonnement.
- WordPress est relativement facile à apprendre, tout en donnant une grande liberté visuelle dans la conception des sites.
