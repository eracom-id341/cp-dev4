Un travail réel utilisant le CMS WordPress. Nous travaillons sur un redesign du site officiel de l'Eracom. Ce nouveau site aura un design visuel plus contemporain, et permettra notamment de mettre en valeur les travaux réalisés dans l'école.

C'est un travail collaboratif entre plusieurs classes.

- Classe ID421 : a conçu un Wireframe et une charge graphique. (semestre passé)
- Classe ID441 : vont créer les Maquettes (Figma)
- Classe ID331 : vont concevoir un design system (Figma)
- **Classe ID341** : va intégrer les maquettes dans WordPress.

Ce travail débutera le 23 février, et se déroulera en trois phases. 

La première phase se déroule sur quatre séances: elle débute le 23 février et se termine le lundi 16 mars.

Pour gérer ce projet collectif, nous utiliserons [une méthodologie Agile nommée Scrum](https://cours-web.ch/agile/).
